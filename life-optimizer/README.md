# Life Optimizer

## Overview
The Life Optimizer is a Python application designed to help users optimize their daily activities and improve their overall well-being. By analyzing various data inputs, the application provides personalized recommendations for physical strain and recovery.

## Installation
To set up the project, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/life-optimizer.git
   cd life-optimizer
   ```

2. Create a virtual environment (optional but recommended):
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute the following command:
```
python src/app.py
```
